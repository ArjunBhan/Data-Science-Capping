The Globetrotters

The globetrotters have created a comprehensive NBA data analysis toolkit. This toolkit consists of

A dashboard providing access to data, data metrics, data visualization, and access to predictive models (you are looking at it now) A fully documented REST API providing access to data and access to predictive models An unsupervised clustering approach to give insight into players and their preformance A neural network utilizing sentiment analysis of player tweets to predict outcomes of games A reinforcement based approach to making a bot capable of betting on NBA games A ensemble based approach using both the neural network and reinforcemnet approach for betting on NBA games An automated data pipeline that loads data into a database, cleanses it, and preforms feature engineering A data queue for a distrubted system of data producers which gives the potential for the database to always be up to date A statistical analysis and validation of the data

All code, data, and the project paper is stored in a github repo https://github.com/NicholasBlaskey/DATA477-capping

The presentation can be found in the Panopto in the iLearn site
